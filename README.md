# Tally Tracker App

A simple app to count and track numbers every day.

---

## Features

* Tap the number or press **Spacebar** to count.
* Set a daily goal and watch your progress.
* View all counts on a calendar.
* Save your counts to a file.
* Load your counts from a file.
* Clear all counts to start fresh.

---

## Pages Guide

### Counter Page

* Tap the main number to add to your count.
* Press **Spacebar** for quick counting.
* Set your daily goal using the input box.
* Reset your count to zero anytime.
* Save your tally when finished for the day.

### Dashboard Page

* See your counts on a monthly calendar.
* Check daily totals and the selected month total.
* Navigate months using the arrows.
* Quickly see your progress toward goals.

### Backup & Restore Page

* **Export**: Save all your counts to a file.
* **Import**: Load counts from a previously saved file.
* **Clear All**: Delete all counts permanently to start fresh.

---

## How to Use

1. Open the app in your browser.
2. Use the **Counter Page** to track daily numbers.
3. Visit the **Dashboard Page** to see totals and history.
4. Use the **Backup & Restore Page** to save or load your counts.

---

## Tips

* **Spacebar** is a quick way to add counts.
* Calendar shows daily totals for easy tracking.
* Always confirm before clearing all data.
